# Asset Allocation Project

Computational Finance project developed during the M.Sc. in Mathematical Engineering at Politecnico di Milano (AY 2025-2026).

## Overview
The project explores different portfolio allocation strategies, including:
- minimum-variance portfolios
- maximum-Sharpe portfolios
- robust and resampled efficient frontiers
- Black-Litterman allocation
- diversification-based portfolios
- PCA-based and CVaR-based portfolio construction
- out-of-sample testing on different market periods

## Structure
- `codes/`: MATLAB scripts and functions
- `reports/`: project reports

## How to run
Open MATLAB, set the `codes/` folder as the working directory, and run:

```matlab
Run_Project
```
